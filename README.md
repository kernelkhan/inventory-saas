# SaaS Inventory Management System

A production-grade SaaS Inventory Management System using the PERN stack (PostgreSQL, Express, React/Next.js, Node.js).

## Features
- **Authentication**: Secure JWT-based login with HttpOnly cookies.
- **Inventory Management**: CRUD operations for products.
- **Analytics**: Real-time sales data visualization.
- **Multi-tenancy**: Designed for scalable usage.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS, Redux Toolkit
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL, Prisma ORM
- **Infrastructure**: Designed for AWS EC2 deployment

## Getting Started

### Backend
1. Navigate to `server` directory.
2. Run `npm install`.
3. Configure `.env` with `DATABASE_URL`.
4. Run `npx prisma migrate dev`.
5. Start server: `npm run dev`.
